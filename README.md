# tree-compoennt

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### example
```
<template>
  <div id="app">
    <KyTree />
  </div>
</template>

<script>
import KyTree from './components/ky-tree/app.vue'

export default {
  name: 'app',
  components: {
    KyTree
  }
}
</script>
```

### Lints and fixes files
```
yarn run lint
```
